# JapaneseLearning-Capture

日语学习资料管理器，可管理日语词汇、语法解释、听力练习、阅读材料等。用于提高日语阅读、口语、听力水平，以应对JLPT（日本语能力测试）。祝各位早日考过N1、N2，加油！
